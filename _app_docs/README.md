# App Docs

Documentazione descrittiva per ogni app Django locale del progetto.

## Come usarla

1. Apri il file dell'app in `_app_docs/<app>.md`.
2. Aggiorna le sezioni quando aggiungi o cambi feature.
3. Usa la sezione "Casi d'uso" per ricordare i flussi reali.
4. Usa la sezione "Debito tecnico / TODO" per fissare i prossimi step.
